## 문제

[백준 9625. BABBA](https://www.acmicpc.net/problem/9625)

## 풀이

## 코드

```
a = []
b = []

a.append(1)
b.append(0)

for i in range(1, 46):
    a.append(b[i-1])
    b.append(a[i-1] + b[i-1])

K = int(input())
print(a[K], b[K])
```
